# Unet-to-erase
手写笔记擦除实验采用U-net结构搭建，数据集为约2000张大小不一的图片，8张图片为一个batch，
训练验证比采用8：2 在预处理过程统一尺寸到512×512，
并进行翻转，角度偏移，调整亮度，对比度，添加噪声等处理提升模型鲁棒性 
U-net采用4层结构，第一层采用，每层解码器与编码器之间采用跳跃连接 
损失函数采用L1loss和ssimloss，优化器为AdamW，并且设置可变学习率 
最终跑完101轮epoch，train_loss为0.0198，psnr为26.1974，ssim为0.9762，可视化效果如图所示 
反思：有些较为细微的笔记仍无法擦除，感觉可以加深层数提取更多精细特征

注意：由于数据集非本人所有，无法开源，如需复现此实验，可以自己收集数据（手写笔记与擦除效果一一对应即可）
